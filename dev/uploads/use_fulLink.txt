Spicetify themes git:
    https://github.com/spicetify/spicetify-themes/blob/master/THEMES.md
        If you want to re-apply themes run:
            - spicetify config => to check if theme or color_scheme are configured
            - spicetify config current_theme THEME => set the current theme
            - spicetify config color_scheme COLOR_SCHEME => set the color scheme
            - spicetify apply => applies current changes
